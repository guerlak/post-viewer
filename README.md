### Utilização para desenvolvimento

## API

- `cd api`

- `yarn`

- Renomeie `.env.example` para `.env` e edite os parametros se necessário.

- `yarn dev:server`

## Client

- `cd client`

- `yarn`

- `yarn start`

Abrir [http://localhost:3000](http://localhost:3000) para visualização do app no browser.
